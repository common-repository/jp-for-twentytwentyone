Customize Twenty Twenty-One theme for Japanese.

== Description ==

Customize Twenty Twenty-One theme for Japanese.
This plugin only works with Twenty Twenty-One and its child themes.

= Features =

* Set the font family for Japanese.
* Noto Sans JP fonts can be applied.
* Font size can be set.
* Add major social icons (LINE, Hatena, Rakuten) in Japan.
* Change the "Powered by" text in the footer.
* Set the content width and the standard maximum width of the content.
* Add the breadcrumb.
* Add the table of contents.
* Add the Scroll Back to Top button.
* Add the sidebar. There is also a widget area that tracks scrolling.
* The table of contents can be automatically inserted into the sidebar (sticky).
* Set the background image of the header.
* jQuery is not used on the front side.

= Restrictions =

* Some features do not work in IE.

== Installation ==

1. Upload the `jp-for-twentytwentyone` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
